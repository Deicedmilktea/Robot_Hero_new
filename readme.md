## 操作说明
### 遥控链路
```plain
 *                                                            *
 *   ------------------------------------------------------   *
 *   |     (上-1) 发射(摩+拨)                 (上-1) 视觉 |    *
 *   |SW_L|(中-3) 摩擦轮                 SW_R|(中-3) 键鼠 |    *
 *   |     (下-2) 全停                        (下-2) 遥控 |    *
 *   |                                                   |    *
 *   |    pitch                               chassis    |    *
 *   |    | ^ |                                | ^ |     |    *
 *   |    | 3 |左摇杆                     右摇杆| 1 |     |    *
 *   | ---     ---                          ---     ---  |    *
 *   |<           2>  gimbal               <           0>|    *
 *   | ---     ---                          ---     ---  |    *
 *   |    |   |                                |   |     |    *
 *   |    |   |                                |   |     |    *
 *   |                                                   |    *
 *   -----------------------------------------------------    *
 *                                                            *
 注：上场模式将右拨杆上更改为yaw轴陀螺仪校正
     遥控模式默认为normal，便于过检录
```
